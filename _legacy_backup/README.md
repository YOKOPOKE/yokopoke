# Yoko Poke House Website

Este proyecto contiene la página de aterrizaje (landing page) para Yoko Poke House.

## Estructura del Proyecto

*   `index.html`: El archivo principal HTML.
*   `styles.css`: Hojas de estilo personalizadas para el diseño "Premium".
*   `script.js`: Lógica de la aplicación (menú, selector de ingredientes, animaciones).
*   `tailwind_config.js`: Configuración de colores y fuentes de Tailwind CSS.

## Cómo Ejecutar

Simplemente abre el archivo `index.html` en tu navegador web de preferencia (Chrome, Edge, Firefox).

No es necesario instalar dependencias ni usar servidores complejos, ya que utiliza librerías via CDN (Tailwind, FontAwesome, AOS).
